### Para el uso de esta webapp necesita lo siguiente: ######
*Instalar proyecto que se llame "modulo3" *
vue create modulo3
*instalar preset Manually*
*seleccionar:*
babel, router, vuex, Css Preprocessors
*Use History mode for router?: Yes*
*Pick Sass/SCSS(with-dart-sass)*
*Where do u prefer placing config for babel,...?
in package.json*
*save this as a preset for future projects?* NO

PARA INSTALAR BOOTSTRAP POR NPM:
npm i bootstrap jquery popper.js

Para instalar FontAwesome:
npm i --save @fortawesome/fontawesome-svg-core
npm i --save @fortawesome/free-solid-svg-icons
npm i --save @fortawesome/free-brands-svg-icons
npm i --save @fortawesome/vue-fontawesome

PARA INSTALAR FIREBASE: 
npm install -g firebase-tools
npm install firebase
npm install firebase vuefire --save
